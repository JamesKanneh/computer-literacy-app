# Computer Literacy CLI App

This is a **menu-driven Python program** designed to educate the young generation on computer literacy.  
Users can learn the basics of computers, access beginner resources, and understand key computer concepts.

## Features
- Add educational topics  
- View existing resources  
- Search resources by keyword  
- Simple text-based menu interface
